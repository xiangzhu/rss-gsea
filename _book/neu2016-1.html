11.1 GSEA Round 1

Last updated: 2016-05-07

Code version: 8be56d426efd3b92bc1e4ecbcbf0677e63d8e492

11.1.1 Fit the null model

I used the grids theta0=(-4.5:0.25:-2)' and h=(0.3:0.1:0.6)' when fitting the null model. I estimated the (unnormalized) log importance weights (“pseudo-likelihoods”) for all 56 sets of (h,theta0) under null.

After normalizing the log “pseudo-likelihoods” (logw.step*) to posterior probabilities (posp.step*), we can see that almost 90.5% posterior mass is concentrated on the settings (h=0.3, theta0=-4) and (h=0.3, theta0=-4.25).

11.1.2 Fit the enrichment model

To perform GSEA, I set (h=0.3, theta0=-4), since the null analysis suggests that almost 100% posterior mass is placed on this setting. For the log-fold enrichment parameter, I use the grid theta=(0:0.02:4)'. The following table lists the GSEA results.
